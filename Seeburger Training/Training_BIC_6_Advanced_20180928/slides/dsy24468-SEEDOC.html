<?xml

This part advises the browser that the following structure is an XML document.

 
version="1.0"

This is the XML versionnumber

 
encoding="UTF-8"

The keyword "encoding" considers which character set should be used. Besides the usual ones as "UTF-8" or "UTF-16" there are further country-specific character sets like ISO-8859-1 or ISO-8859-10.
Overview

The converter can be used for creating all ordinary XML imformation, e.g.

  • XML version
  • XML encoding
  • document type definitions
  • processing instructions, etc. 

Because this information is not available as records or fields in the XML message description, special BIC functions can be used to fill this information. These functions are described below.

We recommend setting the encoding to ISO-8859-1 for common use in Western Europe to enable the receiver of XML files to process special characters (like French accents and German umlauts).
