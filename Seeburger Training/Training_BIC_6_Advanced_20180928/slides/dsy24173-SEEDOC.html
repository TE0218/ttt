How to use arrays

  • They can be created globally or locally like any other variables
  • The type of arrays must be defined when array-variable is created

Syntax:

NAME TYPE [ ] ;

Example:

  • Numeric array: number%[];
  • Alphanumeric array: character$[];

Access to fields of the array always starts at index 0.

  • e.g. number%[0] will return the first value of the array, number%[2] will return the third value of the array
