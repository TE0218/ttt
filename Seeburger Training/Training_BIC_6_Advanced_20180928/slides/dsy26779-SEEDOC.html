openMTDBConnection

Command is used to open a connection to a message tracking database (if not opened yet). The connection information has to be saved in the SmartiDS datasource.

Read from additional table tBISMT_Document. Use the primary key to identify the database record.

Read from additional table readMTAdditionalDetailRecord. Use the primary key to identify the database record.

openMTDBConnection()

Description

This command is used to open a connection of a message tracking database (if not opened yet). The connection information has to be saved in the SmartiDS datasource. Please refer also to the topicFunctions for Message Tracking Databases.

Parameter(s)

This command does not expect any parameters.

Return Value

This command does not return a value.

Warning(s) / Error(s)

If t he connection to the database cannot be established, the error

  • 2600

is created.

Example 
openMTDBConnection();
readMTRecord Syntax

readMTRecord(primaryKey)

Description

Reads a record from the master message tracking database table (tBISMT_Workflow). The database has to be connected over the openMTDBConnection command before. Please refer also to the topicFunctions for Message Tracking Databases.

Parameter(s)

Parameter

Type

Description

primaryKey

$

The primary key of the database record, which is read.
Return Value

This command does not return a value.

Warning(s) / Error(s)

If the connection cannot be established, an SQL error occurs or the record is not found, the error

  • 2604

is created.

Example 
local primKey$;
//key has to be set
readMTRecord(primKey$);
