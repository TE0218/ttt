CSV - Tutorial 1

Instruction

Illustration

  1. Start BIC Mapping Designer and create a new project named CSVAdapter.

Check chapter "How to Operate BIC Mapping Designer" in Training BIC 6 Basics if you need further assistance.

  1. Create a new CSV message named CATALOGUE_CSV and open it with a double-click.

  1. Create a new record and define the record details. Rightclick on message CATALOGUE_CSV and select Record.

  • Name = RECORD
  • Mandatory = yes
  • Min Occ = 1
  • Max Occ = 999

  1. Create a new field CAT_VERSION below RECORD. Rightclick on RECORD and select Field.

    Define general field details

  • Name = CAT_VERSION
  • Title = Catalogue Version

  1. Define data field details

  • Mandatory = No
  • Min Occ = 0
  • Max Occ = 1
  • Type = AN..
  • Minimal length = 0
  • Maximal length = 3

  1. Repeat the steps 4 and 5 according to the table in the exercise. You can doublecheck your result with file msg_CATALOGUE_CSV_ solution.xml in the classmaterial.
