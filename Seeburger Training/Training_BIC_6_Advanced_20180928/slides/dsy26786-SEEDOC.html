BIS:MT - Tutorial 3

Instruction

Illustration

  1. Extend the mapping you have written before with the update statement and read out again. Set a clearMTRecord between your commands

 
local l_Output$;

executeMTSQLCommandWithMultiResult("select * from TBISMT_WORKFLOW");
for Loop_1% = 0 to 10000
    if hasMoreMTRecords()
         nextMTRecord();
         l_Output$ = getMTField("CPARTNER");
         traceln(l_Output$);
    else
        exitfor;
    endif
next

clearMTRecord();
executeMTSQLCommand("UPDATE TBISMT_WORKFLOW SET CPARTNER= 'Your Company' WHERE CPARTNER= 'DEMO'" );


executeMTSQLCommandWithMultiResult("select * from TBISMT_WORKFLOW");
for Loop_1% = 0 to 10000
    if hasMoreMTRecords()
         nextMTRecord();
         l_Output$ = getMTField("CPARTNER");
         traceln(l_Output$);
    else
        exitfor;
    endif
next
  1. Check your result in the logfile, BIS:MT and also validate it against the trainer solution.
