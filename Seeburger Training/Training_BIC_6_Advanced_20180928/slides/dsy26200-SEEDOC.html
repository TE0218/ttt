BICMD database structures - overview

  • it’s possible to access the database tables via BICMD SQL structure

  • each record at the structure describes a table at the DB

  • it’s also possible to create a hierarchy with one root table and corresponding child tables

  • primary and foreign keys are used to realize the relation between the tables

  • the connection to the DB is realized via the SQL message structure

    • all connection information are part of the structure

    • contains update information for read access at the DB

  • the BICMD JDBC wizard is used to create the SQL structure
