Walkthrough

Characteristics

  • Child and Error mappings are used to start another mapping out of a running mapping (main mapping)

  • Possible use cases:

    • Incoming file should be mapped to different destinations

    • Incoming file contains different message formats of the same message type

e.g.: incoming ANSI file with 830 and 862 or incoming EDIFACT file with DELFOR and DELJIT
