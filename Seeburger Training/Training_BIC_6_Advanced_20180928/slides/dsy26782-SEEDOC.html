Direct Commands - 1

Command

Example

Description

openMTDBConnection

 
openMTDBConnection();

Command is used to open a connection to a message tracking database (if not opened yet). The connection information has to be saved in the SmartiDS datasource.

executeMTSQLCommand

 
executeMTSQLCommand("SELECT count(*) AS counter
	FROM tBISMT_Worklfow
	WHERE ( cFree7 = 'SEEBURGER')");

Is used to execute an SQL command in the Message Tracking database. Get one result from your SQL- command back.

executeMTSQLCommandWithMultiResult

 
executeMTSQLCommandWithMultiResult
("select * from tBISMT_Worklfow");

Executes an SQL command in the message tracking database. This command can be used for getting more then one record as a result.

Use a Loop to run through all results, for checking the next values use

  • hasMoreMTRecords

  • nextMTRecord
For more details please refer to openMTDBConnection Syntax

openMTDBConnection()

Description

This command is used to open a connection of a message tracking database (if not opened yet). The connection information has to be saved in the SmartiDS datasource. Please refer also to the topicFunctions for Message Tracking Databases.

Parameter(s)

This command does not expect any parameters.

Return Value

This command does not return a value.

Warning(s) / Error(s)

If t he connection to the database cannot be established, the error

  • 2600

is created.

Example 
openMTDBConnection();
executeMTSQLCommand Syntax

executeMTSQLCommand (sqlString)

Description

Is used to execute an SQL command in the Message Tracking database. The database has to have been connected with the openMTDBConnection command. Please refer also to the topic Functions for Message Tracking Database, particularly the command executeMTSQLCommandWithMultiResult .

Parameter(s)

Parameter

Type

Description

sqlString

$

The SQL command to execute.
Return Value

This command does not return a value.

Warning(s) / Error(s)

If the connection cannot be established or an SQL error occurs, the errors

  • 2606
  • 2607

are created.

Example 
local count%;
executeMTSQLCommand("SELECT count(*) AS counter
	FROM tBISMT_Worklfow
	WHERE (cFree6 = ‘" & ISA_Sender$ & "’) AND ( cFree7 = ‘" & ISA_Receiver$ & "’)");;
count% = getMTField("counter");
clearMTRecord();
executeMTSQLCommandWithMultiResult Syntax

executeMTSQLCommandWithMultiResult (sqlString)

Description

Executes an SQL command in the message tracking database. This command can be used for getting more then one record as a result. Please refer also to the other functions for the Message Tracking Database, especially the commands executeMTSQLCommand , hasMoreMTRecords, nextMTRecord, getMTField and clearMTRecord . Please refer also to the topicFunctions for Message Tracking Databases.

Parameter(s)

Parameter

Type

Description

sqlString

$

The SQL command to execute.
Warning(s) / Error(s)

If the connection cannot be established or an SQL error occurs, the errors

  • 2611
  • 2612

are created.

Example 
local i%;
local test$;
executeMTSQLCommandWithMultiResult("select * from MasterDB");
for i% = 0 to 10000 step 1
    if hasMoreMTRecords()
         nextMTRecord();
         test$ = getMTField("TEST");
        …
    else
        exitfor;
    endif
next
