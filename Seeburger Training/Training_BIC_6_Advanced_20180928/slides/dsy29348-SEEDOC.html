SAP Java Connector (JCo)

  • SAP JCo is a toolkit for building applications written in Java and communicating with R/3 systems over the RFC interface.

  • In order to access the BAPI functions on a SAP system, the com.seeburger.osgi.com.sap.sapjco3.jar (SAP-JCO-API Version 3), and corresponding sapjco3.dll files from SAP are needed (version 3.06 or higher).

  • They need to be downloaded from SAP Marketplace and added to the classpath. During the exercises we will integrate the SAP JCO libraries into BIC Mapping Designer.

The previous sapjco.jar (Version 2) is no longer supported!

Please always check the latest release notes of your BIS installation media.
