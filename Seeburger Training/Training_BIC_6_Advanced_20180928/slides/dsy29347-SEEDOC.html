What is a BAPI?

  • BAPI stands for Business Application Programming Interface.

  • BAPI is a library of functions that are publicly accessible through an interface from an external system

  • In the SEEBURGER mapping environment we use BAPI functions to read or create objects directly in SAP.

The BAPI functions can be called via RFC (remote function protocol) from an external program / system.

For more information about BAPI, please have a look at help.sap.com
