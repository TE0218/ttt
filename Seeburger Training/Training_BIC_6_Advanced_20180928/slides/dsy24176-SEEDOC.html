How to use mixed types

  • They can be created globally or locally like any other variables
  • The type of mixed type must be defined when mixed type-variable is created

Syntax:

NAME TYPE { } [ ] ;

Access to fields of the hashmap is a combination of KEY/VALUE and INDEX.

  • e.g. hashmap${abc}[1] will return VALUE that belongs to the KEY abc and INDEX 1.
