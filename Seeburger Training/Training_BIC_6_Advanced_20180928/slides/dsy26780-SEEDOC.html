openMTDBConnection()

Description

This command is used to open a connection of a message tracking database (if not opened yet). The connection information has to be saved in the SmartiDS datasource. Please refer also to the topicFunctions for Message Tracking Databases.

Parameter(s)

This command does not expect any parameters.

Return Value

This command does not return a value.

Warning(s) / Error(s)

If t he connection to the database cannot be established, the error

  • 2600

is created.

Example 
openMTDBConnection();
setMTField Syntax

setMTField(key, value)

Description

It is used to fill a field/column in the current Message Tracking database record. The database has to be connected over the openMTDBConnection command before. Please refer also to the topicFunctions for Message Tracking Databases.

Parameter(s)

Parameter

Type

Description

key

$

The name of the column to fill.

value

$

The new field/column value.
Return Value

This command does not return a value.

Warning(s) / Error(s)

This command does not raise any warnings or errors.

Example(s): 

setMTField("Cfree4", "BMW");
writeMTRecord Syntax

writeMTRecord(primaryKey, parentPrimaryKey) 

Description

It is used to write the current message tracking database record to the master message tracking database table (tBISMT_Workflow). The database has to be connected over the openMTDBConnection command before. If the record is already present, an update (otherwise an insert) is performed. Please refer also to the topicFunctions for Message Tracking Databases.

Parameter(s)

Parameter

Type

Description

primaryKey

$

The primary key of the database record.

parentPrimaryKey

$

The primary key of the parent record. Also null or "" is applicable.
Return Value

This command does not return a value.

Warning(s) / Error(s)

If the connection cannot be established, an SQL error occurs or no current record is present, the error

  • 2602

is created.

Example(s): 

local primKey$;
local parentKey$;
//keys have to be set
writeMTRecord(primKey$, parentKey$);
writeMTDetailRecord Syntax

writeMTDetailRecord(primaryKey, masterKey) 

Description

This command is used to write the current message tracking database record to the detail message tracking database table (tBISMT_Documents). The database has to be connected over the openMTDBConnection command before. If the record is already present, an update (otherwise an insert) is performed. Please refer also to the topicFunctions for Message Tracking Databases.

Parameter(s)

Parameter

Type

Description

primaryKey

$

The primary key of the database record.

masterKey

$

The primary key of the master database record. Also null or "" is applicable.
Return Value

This command does not return a value.

Warning(s) / Error(s)

If the connection cannot be established, an SQL error occurs or no current record is present, the error

  • 2603

is created.

Example

local primKey$;
local masterKey$;
//keys have to be set

writeMTDetailRecord(primKey$, masterKey$);
