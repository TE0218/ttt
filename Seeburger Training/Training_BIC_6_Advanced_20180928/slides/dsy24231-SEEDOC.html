Procedures - Tutorial 1

Instruction

Illustration

  1. Import the mapping map_I_SHPMNT05_E46C_ to_I_ADDRESSES_INH_ template.zip.bicmd from the classmaterial.

Check chapter "How to Operate BIC Mapping Designer" in Training BIC 6 Basics if you need further assistance.
  1. Go to record program DC1 and define additional variables.
 
global procedure_flag%;
  1. Go to the "Procedure source" tab and click the "New" button. New window will be opened.

Add new local procedure

  1. Define procedure properties:
  • Name = check_role
  • Description =
    returns:
    0 = wrong role;
    1 = OK
  • Return value type = % (numeric)
  1. Go to parameter tab.

Add new local procedure - general properties

  1. Use the "Add" Button to add one new line for one new parameter. Doubleclick the new line.

Add new local procedure - parameter properties

  1. Define parameter details:
  • Type = Alphanumerical
  • Name = role

Add new local procedure - edit parameter

Add new local procedure - parameter properties (role$)

  1. Add procedure code to "Procedure source" tab, using role information as an input parameter and result Code as an output parameter.
 
local result%;
result% = 0;

if role$ = "OSO"
    result% = 1;
endif
if role$ = "AG"
    result% = 1;
endif
if role$ = "WE"
    result% = 1;
endif
if role$ = "SP"
    result% = 1;
endif
if role$ = "OSP"
    result% = 0;
endif

exitproc(result%);
  1. Go to record program FileEnd and change the mapping.
 
local LOOP1%;

for LOOP1% = 1 to COUNT_RM1%
    procedure_flag% = check_role(PARTNERID$[LOOP1%]);
    if procedure_flag% = 1
        copy PARTNERID$[LOOP1%] to ADDRESS.ROLE:NAME;
        copy NAME$[LOOP1%]      to ADDRESS.NAME:NAME;
        copy STREET$[LOOP1%]    to ADDRESS.STREET:STREET;
        copy ZIP$[LOOP1%]       to ADDRESS.ZIP:ZIP;
        copy CITY$[LOOP1%]      to ADDRESS.CITY:CITY;
        copy COUNTRY$[LOOP1%]   to ADDRESS.COUNTRY:COUNTRY;
    endif
next

  1. Have a look at the destination-file and see how the local procedure worked.

You can compare your results with the attached trainer solution.
