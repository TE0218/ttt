XML handling in Mapping Designer - 1

Representation of XML elements in  BIC Mapping Designer

  • All elements are records and all attributes are fields.

  • Elements that contain text values are records with a special field value. This field allows to access the text value of the element. This also applies for elements that contain values and other elements or attributes.

  • Attributes with the name value have to be quoted by means of a backslash ("\").

  • Only elements that contain sub-elements or attributes are shown as records.

  • Elements that contain only text values are shown as fields with the name corresponding to the element name.

  • An element that contains both sub-elements and text values (mixed element types) are shown as both, record and field.

  • Inheritance is marked with a green arrow

  • However, if XML fields and elements are used in addresses, the BIC runtime notation is to be used.
