XML commands in Mapping Designer - 2

Command

Description

Details for execution

setXMLVersion

Modifies the XML version in the XML header.

Should be set in "Destination options" (or in New Mapping)
default "1.0"

setXMLEncoding

It creates a value in the version tag but does not effect any character set or code page conversion. For this set also setDestinationEncoding

Should be set in "Destination options" (or in New Mapping)

setDestinationRegularExpressionCheck

Enables or disables the regular expression check for destination message fields.If a field does not match, an error is reported.

Should be set in "Destination options" (or in New Mapping)
valid options are "on" or "off"

The regular expression can be found in field properties
