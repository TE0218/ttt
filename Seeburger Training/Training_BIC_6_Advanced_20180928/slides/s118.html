Walkthrough

Why BIS:MT

  • The BIS Message Tracking is a functionality of the standard BIS:Portal.

  • It provides display and search functions for data within messages. With the BIC message tracking commands, information from the Message Tracking database can be read, and new information can be written during conversion.

  • The message tracking command reuses the pre-defined SEEBURGER data source SmartiDS.

Demonstration
