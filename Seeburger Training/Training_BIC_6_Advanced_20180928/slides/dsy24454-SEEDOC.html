XML commands in Mapping Designer - 1

Command

Description

Details for execution

ignoreXMLNamespacePrefixes

Specifies if the namespace prefixes should be ignored in the XML tags of the XML source files.

The source message description MUST NOT contain any namespace prefixes.

Should be set in "Source options" (or in New Mapping)
valid options are "on" or "off" - default "off"

enableSourceXMLEmptyFieldCheck

Enables the check of empty XML tags in the source message.

Should be set in "Source options" (or in New Mapping)
valid options are "on" or "off" - default "off"

Based on error/warning level the mapping will run into error if a XML tag is empty.

setSourceRegularExpressionCheck

Enables or disables the regular expression check for source message fields.If a field does not match, an error is reported.

Should be set in "Source options" (or in New Mapping)
valid options are "on" or "off" - default "off"

The regular expression can be found in field properties
