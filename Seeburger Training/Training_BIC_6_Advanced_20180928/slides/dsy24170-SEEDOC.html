What is an array? - 2

Example:

Difference vector/matrix

  • Arrays can be initialized so the size is predefined and the memory is reserved

    • test$[5]

    • test$[5][5]

  • If you don't know the count of entries to be stored, you can initialize it without a predefined value

    • test$[]

    • test$[][]

Be careful if you access a non existent field it will be automatically created.

The initialization of the array size does only take effect in mapping runtime version V1 (Only used in old BIC 5 compatible mappings, can be set in the Mapping properties). In Runtime version V2 (BIC 6) the size initialization is not needed and can be exceeded without warnings.
