Tradacoms Characteristics - 1

  • Each Tradacoms message consists of segments. Each segment consists of elements.

  • STX/END segments used instead of Edifact UNB/UNZ

  • MHD/MTR segments instead of Edifact UNH/UNT

  • The segment tag delimiter is an '='

  • The default EDIFACT UNOA separators are used:

Default character

Meaning

+

data element separator

:

component data element separator

.

decimal notification

?

release character

'

segment terminator

=

 segment tag delimiter
