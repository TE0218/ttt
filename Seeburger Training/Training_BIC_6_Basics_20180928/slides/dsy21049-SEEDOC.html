How to use a MapFile - openMapFile

The OpenMapFile command opens a map file and allows the access of the mapfile command.

There are the following types:

  • openMapFile(ChannelName, FileName, SourceColumn_No,...DestinationColumn_No);

  • openMapFile(ChannelName, FileName, SourceColumn_No,...DestinationColumn_No,Separator);

  • openMapFile(ChannelName, FileName, SourceColumn_No,...DestinationColumn_No,Separator, DefaultValue);

// Open Mapfile Channels
openMapFile("Country","Country_to_Currency.cde", 1, 2,",","unknown");
openMapFile("Currency","Country_to_Currency.cde", 1, 3,",","unknown");
openMapFile("Continent","Country_to_Currency.cde", 1, 4,",","unknown");
