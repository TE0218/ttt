Extended functionalities

For more complex scenarios or to meet reporting requirements there are further command sets.

Access a database manually

connectDB, openlookupDB, …

Transcode values (mapfile / database table)

openmapFile, openmapDB, mapFile, mapDB, …

Interaction with BIS processes

getInputValue, setOutputValue, ...
Write to Message Tracking database

writeMTRecord, …
Integration of SAP Business Objects (BAPI cal l)

createBAPICall, executeBAPICall, ...
Embedding of Java code

Not recommended from a support perspective but it may be the only way to solve very special requirements.

A simple code example shows the usage of Java code to determine a UNIX timestamp:

local time$;
#javastart
   _StrVar_TIME.setString(System.currentTimeMillis());
#javaend
