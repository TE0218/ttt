Instruction

Illustration

  1. Use the Create new project icon (or alternatively menu entry File --> Create new project).

Create new project

  1. Choose a project name and click on OK.

Project Properties

  1. Select the newly created project. Click on the Import button (or alternatively menu entry File --> Import file).

Import File

  1. Navigate to the folder where you have stored the class material. Select the first message file and click on Open. Repeat steps 3 and 4 for the second message and also the mapping file.

Select file for import

  1. Open the mapping (double click) and become familiar with how it works.

Mapping editor

  1. Compile the mapping (use the icon or alternatively Build --> Compile).

Compile mapping

  1. Open the test environment.

Open Test environment

  1. Creatre a new Mapping configuration

Create new Mapping configuration

  1. Choose a source file and define the path and name of the conversion result.

Source and Destination File

  1. Convert the source file (use the icon or alternatively the menu Build --> Convert).

Convert the source file

  1. Open the destination file.

Open destination file

  1. There is a Compile, Convert and Show button to combine these steps.

Compile, Convert and Show
