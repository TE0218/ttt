Tradacoms Example

STX=ANA:1+5000000000000:AAA+5010000000000:BBB+070315:130233+000007+PASSW+ORDHDR+B'
MHD=1+ORDHDR:9'              START ORDER HEADER
TYP=0430+NEW-ORDERS'
SDT=5010000000000:000030034'
CDT=5000000000000'
FIL=1630+1+070315'
MTR=6'                       END ORDER HEADER 
MHD=2+ORDERS:9'              START ORDERS 
CLO=5000000000283:89828+SOUTHWEST DEPOT' 
ORD=70970::110907'
DIN=070321++0000' 
OLD=1+5010210000000++:00893592+12+60++++DELICIOUS ROLLS:4PACK' 
OTR=1' 
MTR=7'                       END ORDERS 
MHD=3+ORDTLR:9'              START ORDER TRAILER 
OFT=1' MTR=3'                END ORDER TRAILER 
END=3'

Get familiar with the Tradacoms elements and characteristics on the basis of this example:

  • The interchange control reference is "000007"

  • The customer's order number is "70970" and was placed on September 7th 2011

  • The Total Number of Orders in this message is "1" (OFT=1...). The Order Details (ORDERS) consist of 7 segments (MTR=7).

  • The Tradacoms file consists of 3 messages (END=3)
