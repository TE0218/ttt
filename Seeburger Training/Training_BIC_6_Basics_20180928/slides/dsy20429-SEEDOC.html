Auto-Completion

In order to ensure mapping is easy and efficient, there is an auto-completion feature which assists the user during code creation

  • Ctrl+Space

    • Opens a list of all available commands and variables

  • Ctrl+Shift+F

    • Opens a list of all available functions (commands such as isNumeric)

  • Ctrl+Shift+C

    • Opens a list of all available commands (such as copy, if etc.)

  • Ctrl+Shift+V

    • Opens a list of all available variables

  • Typing a space after the keyword copy will open a list of all available functions and variables.

  • Press ENTER or SPACE to use the selected entry / Press ESC to close the list
