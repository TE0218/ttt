Help Function

  • The BIC Mapping Designer offers a help function with a full list of all available commands. It can be accessed via the menu bar (? --> Help). Each function is described in a consistent way:

    • Syntax

    • Description

    • Parameters

    • Return Value

    • Possible Warning(s) / Error(s)

    • Example  

  • A command reference is also available in PDF format. It is located in your BIS installation folder: <BIS-ROOT>\software\spm-repository\doc\bic\m_BicMDCommandReference_en.pdf

  • If you encounter a command in the mapping code which you need help for just select it and press F1.

  • There is also a short description for every command if you position the mouse cursor over it (mouse-over).

  • Be aware that some commands exist more than once and have the same name but a different number of parameters. Example:

    • mid(string, from) vs. mid(string, from, number)
