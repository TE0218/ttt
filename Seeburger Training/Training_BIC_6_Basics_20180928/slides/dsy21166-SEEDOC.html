Tradacoms BIC Commands

  • The END segment contains the number of transmissions (MHD Segments) in the message.

  • BIC offers the functionality to automatically count and set the values when creating a Tradacom message. The easiest way is to use the field properties (tab Codes & counter)

    • Segment counter in END.NMST

      • Counted record: MHD / Reset records: STX
