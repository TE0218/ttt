Definition of precompiled mapping

  • Precompiled mappings are translations from the origin XML mapping structure to an intermediate format 
  • the intermediate format is called „.jcn“ 
  • the jcn File isn‘t editable
  • jcn mapping files aren‘t importable into BIC MD

Since 6.3.5Q4 it's possible to test precompiled mappings in the test environment of BIC Mapping Designer. This function may be helpful to test a process Mapping together with a precompiled Partner Mapping (SEEBURGER Standard Mapping).
