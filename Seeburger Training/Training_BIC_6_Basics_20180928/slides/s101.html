What is a MapFile / MapDB?

  • The MapDB / MapFile handling provides an easy access for data stored in flat files or database tables.

  • Such external data is expected to be available as a set of key/value pairs.

  • Seeburger provides a BIS internal and logical MapDB that replaces external MapDB databases.

Example: A mapping reads some country name abbreviations from the input and should map therefore some additional information to the output. Then the mapping requires some additional key/value pairs like this:

MapFile

MapDB

 
! This map file maps two-letter country
! codes to currency codes and geographical locations.
! country code | country name | currency | continent
DE,Germany,EUR,Europe
US,United States of Americe,USD,North America
RUS,Russia,RUB,Europe and Asia
