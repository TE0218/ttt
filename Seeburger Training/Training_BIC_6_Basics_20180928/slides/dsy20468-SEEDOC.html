Known Restrictions

  • The BIC MD is shipped as a 32-bit application only and can only be installed on Windows operating systems.

  • Do not use the suggested installation path for 64-bit Windows-based systems (C:\Program Files(x86)\seeburger\BIS\), because it contains special characters (brackets and a blank). It is recommended that you use a short base path for the installation. Select a base directory on a non-system drive, e.g. D:\BICMD635.

  • It is not allowed to select the folder which contains the BIS 6 server installation (BIS 6 installation directory)!

  • The BICMD installs a signed version of the program. In Windows 7 this can lead to a window warning after the installation that the program was not installed correctly. If this warning occurs after the last step, which shows Install Complete

    • Click Done to quit the installer and activate the Done button.
