BIC MD GUI - Preferences

  • Menu bar --> Window --> Preferences

  • Allows to customize different kind of parameters:

    • Mapping view preferences: Font size, syntax coloring, drag&drop properties, code-completion

    • Paths and files: Standard path setting, workspace settings

    • Datasources: Define datasources (databases...)

    • Editors: Define editors to be used for text or XML messages

    • Compiler warnings and errors: Some compiler messages may be configured to be interpreted as error, warning or to be ignored
