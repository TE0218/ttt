Precompiled Mappings - Excercise 2

The precompiled mapping can either be imported/activated in BIS using BIS Front-end. For testing purposes it can also be used in the BIC MD test environment.

Please copy the folder structure of the before delivered precompiled mapping to <Workspace>\src\jcn. The sub-folder jcn has to be created manually. Afterwards you can convert a test message inside the test environment of the Mapping Designer.
