ANSI X.12 BIC Commands

  • The SE segment contains the number of segments in the message. The GE segment contains the number of messages in an interchange file.

  • BIC offers the functionality to automatically count and set the values when creating an ANSI X.12 message. The easiest way is to use the field properties (tab Codes & counter)

  • Segment counter in SE.143

    • Counted record : ALL / Reset records: ST

    • Message counter in GE.479

    • Counted record ST / Reset records: GS
