Numeric commands - 2

  • The maximum accuracy of numeric variables and numbers within BIC is 13 (digits). If more digits are used, the value is rounded to 13 digits.

  • Big decimal variables and functions offer a quasi-infinite accuracy. However you must use the special big decimal functions and the performance is low. So the following functions should only be used if necessary. They replace the standard commands and functions:

    • absBig

    • addBig / subBig

    • compareBig

    • divideBig / multiplyBig

    • minBig / maxBig

    • negateBig

    • roundBig
