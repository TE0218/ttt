BIC Mapping Designer Installation - Exercise

  1. Install the BIC Mapping Designer on your local hardware.

    • Mount the provided ISO file  Z:\_CD_Source\02_BIS\BISDevStudio-6.5.2_SP52.iso using Virtual Clone Drive.

    • After mounting the iso file you can start installation from drive E:\

  2. Import the license file during the installation process.

    • Import Z:\_CD_Source\02_BIS\IntranetLicense.BIC_MD6.5.2.lc

  3. Activate <BICMD-ROOT>\sys\bic_mapping_designer.ini for Java 64bit as you work on a 64bit environment.

  4. Start the BIC Mapping Designer the first time.

  5. Create a new workspace Training_BIC_Basics where all your data will be stored. In the most screens you can use the default settings.
