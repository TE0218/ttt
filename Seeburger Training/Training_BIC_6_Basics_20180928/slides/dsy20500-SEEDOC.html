EDIFACT Structure

  • UNA separator declaration (optional, default value = UNA:+.? ' )

  • UNB Interchange header, contains sender/receiver information

    • UNG Functional group header (optional)

      • UNH Message header, contains message type and version

      • UNT Message trailer, contains the number of segments

    • UNE Functional group trailer (optional)

  • UNZ Interchange trailer, contains the number of messages
