BIC Mapping Designer

  • Independent piece of software which is usually installed on a desktop computer

  • Allows the creation and testing of message structures, mappings and global procedures

  • Saves files locally as (msg/map/gpc)*.xml files

  • Exchange of objects is possible via import/export to file system or repository

  • For using the repository function a connection to a running BIS system is required

  • To use a mapping in a BIS environment it has to be exported to the repository and needs to be activated. Afterwards it can be deployed in the BIS Front-end ("Mapping Management").
