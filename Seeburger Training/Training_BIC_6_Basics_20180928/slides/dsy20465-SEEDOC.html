BIC Mapping Designer Installation - Tutorial

Instruction

Illustration

  1. Click the Next button.

Introduction

  1. Enter or select the folder where you like to install BIC Mapping Designer.

Attention: It is not allowed to select the folder which contains the BIS 6 server installation (BIS 6 installation directory)!

Note: Do not use the suggested installation path for 64-bit Windows-based systems (C:\Program Files(x86)\seeburger\BIS\), because it contains special characters (brackets and a blank). It is recommended that you use a short base path for the installation. Select a base directory on a non-system drive, e.g. C:\BICMD652.

Choose Install Folder

  1. Select which edition from the Mapping Designer you would like to install. Also check the option Install Debugger, if you want to install the new Debugger. The Mapping Debugger helps the user to analyze and test mappings during runtime.

Installation Type

  1. Click the Install button.

Pre-Installation Summary

  1. The BIC Mapping Designer will be installed.

Installation Process

  1. The License Manager opens. Here you can create a license request or import an existing license file.

License Manager

  1. After completion of the BIC Mapping Designer installation, installation-related information is displayed.

Finish Installation

  1. Activate Java 64bit for BIC Mapping Designer. Go to <BICMD-ROOT> \sys\ and rename bic_mapping_designer.in_64 to bic_mapping_designer.ini.

  1. Start the Mapping Designer either by using the start menu entry or by opening the file <BICMD-ROOT>\bin\mapping_designer.exe

  2. We use the option to create a new workspace as follows.

Create new workspace

  1. Set your workspace location as shown in the screen on the right and click on Next.

Set workspace location

  1. Define the runtime location as shown on the screen on the right (default) an click on Next.

Set runtime and build location

  1. Copy the configuration settings from the Default workspace and click on Next.

Copy configuration settings

  1. Please use the default settings for the workspace folder configuration an click on Next.

Workspace folder configuration

  1. Don't import projects from the repository and click on Next.

Import projects from repository

  1. Don't import projects from workspaces and click on Finish.

  2. After your click on Finish the BIC Mapping Designer will start with your new created workspace Training_BIC_Basics.
