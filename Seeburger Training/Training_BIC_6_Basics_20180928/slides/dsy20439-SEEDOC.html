Options - 1

  • There are many options which influence the behaviour of a mapping and its handling of source and destination message.

  • The settings can be done in the mapping code by using the corresponding commands. However it is recommended to use the option tabs.

  • There are three kinds of options:

Global mapping options

These settings apply to the mapping itself. There is no dependency to the used message format.

  • Examples: error handling, syntax check settings, ...

    • setErrorMapping, ignoreAllErrors, modifyWarningLevel, ...
