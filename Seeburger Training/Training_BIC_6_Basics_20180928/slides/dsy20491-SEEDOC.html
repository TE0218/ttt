BIC MD GUI - Exercise 1

A SEEBURGER consultant sent you a new mapping consisting of three files. Get the mapping ready to be used in the mapping designer environment.

The class material contains the needed message structures, mapping and testfiles.

Before starting to import the different objects please unzip the classmaterial zip-file first.

  • Create a new project.

  • Import the two message structures (filename starts with "msg").

  • Import the mapping file (filename starts with "map").

  • Open the mapping and have a look at the different parts of the BIC Mapping Designer GUI.

  • Compile the mapping. Please remember that there are different ways to compile the mapping (menu selection, icon bar, keyboard shortcut).

  • Convert the provided input file by using the test environment. Check the result. How can you execute the steps compile, convert and show with only one mouse click?
