ANSI X.12 Exercise

Create a mapping from ANSI X.12 to CSV (Baan) format. Follow the instructions of how to use the different record programs.

The class material contains the required message structures and testfiles.

After you have finished your mapping you can test it in the test environment. While having a look at the baan output file using the internal message editor it's necessary to set the element delimiter "|" in the properties of the message editor as well.
