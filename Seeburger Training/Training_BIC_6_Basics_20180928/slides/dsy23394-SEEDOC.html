INHOUSE - Exercise 1

Extend an existing message description

  • Create a new project Training_BIC_6_Basics_Inhouse within the BIC MD.

  • Import the message structure msg_PURCHASE_ORDER.xml from the classmaterial. 

  • In the current version of the message there are only fields of the document header. The position details have to be added to the INHOUSE message. Details can be found in the description below.

  • Validate your message description with the solution provided in the class material. You can use the function compare message description.
