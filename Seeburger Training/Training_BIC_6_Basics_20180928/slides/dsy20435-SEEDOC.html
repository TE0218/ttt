Numeric commands - 1

  • Simple calculations can be computed as such:

Addition:

result% = 1 + 2;

Subtraction:

result% = 2 - 1;

Division:

result% = 2 / 2;

Multiplication:

result% = 2 * 2;

  • Numeric commands:

Command

Example

Result

multiDigitRound

multiDigitRound(1234.890445, 3);

1234.891

round

round(1234.890445, 3);

1234.890

isNumeric

isNumeric("123.1"); / isNumeric("A");

1 (= true) / 0 (= false)

Explanation for the different results of the two round functions:

  • multiDigitRound uses more than one iteration. It rounds digit by digit starting with the last digit.

  • round inspects only one more digit than the required number of digits in the result: 1234.8904 --> 1234.890
