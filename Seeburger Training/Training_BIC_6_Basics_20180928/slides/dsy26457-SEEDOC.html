Precompiled Mappings -Tutorial 2

Instruction

Illustration

  1. Copy the folder structure of the precompiled mapping to <Workspace>\src\jcn.The sub-folder jcn has to be created manually first.

In the previous exercise the precompiled mapping was exported to C:\Precompiled_Mapping
  1. Start BIC Mapping Designer and open the Test environment.

Please don't select a project in the project explorer.

  1. Setup a new mapping configuration and select the precompiled mapping (jcn) as first mapping.

Create new mappings configuration

  1. Afterwards please choose the testfile from the classmaterial: SEEDELFOR_ORDERS.xml

  2. Click on compile, convert, check the destination file and the logfile.

    The conversion can only succeed if the maptable has been setup according to modul Mapfile and MapDB Handling.
