ANSI X.12 Characteristics - 2

  • ANSI X.12 is a data format complying to ASC X12 standards.

  • Unlike EDIFACT, each message type has a specific number instead of a name, e.g. 810, 850, 856 etc. 

    • Further, every new release has a new version number, i.e. 4010, 4020, 4030, 5010, 6010 etc.

  • Each message opens with an envelope (ISA, GS, ST) and closes with a counter envelope (SE, GE, IEA). 

    • These envelopes contain specific information regarding trading partners (Sender ID, Receiver ID etc.)

  • Task specific segments are contained within the envelope.

    • They are identified by segment specific qualifiers.
