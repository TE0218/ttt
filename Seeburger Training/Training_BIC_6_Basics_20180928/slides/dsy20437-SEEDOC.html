Conditional Statements

Command

Example

if - else (optional) - endif

Example 1: 

if Action$ &= „Addition“
   counter% = counter% + 1;
endif

Example 2 (with an alternative action): 

if Action$ &= „Addition“
   counter% = counter% + 1;
else
   counter% = counter% - 1;
endif

selectcase - case - caseelse (optional) - endselect 

selectcase Action$
   case „Addition“:       counter% = counter% + 1;
   case „Subtraction“:    counter% = counter% - 1;
   caseelse:              counter% = 0;
endselect
