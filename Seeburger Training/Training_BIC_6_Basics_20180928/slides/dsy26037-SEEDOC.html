MapDB in BIS Front-end vs. BIC Mapping Designer

Below you will find a comparison of the MapDB tables while looking at BIS Front-end and the BIC Mapping Designer.

BIS Front-end

BIC Mapping Designer

  • In BIS Front-end you directly look at the logical MapDB which is used in production.

  • You have on single view to access the logical MapDB
    • Logical table name holds the link to the original mapfile
    • Logical table column names holds the headlines of the mapfile
    • Logical table data holds the detailed information of the mapfile itself.

  • For BIC Mapping Designer we ship a local H2 database where we can import the mapfiles locally.

  • The logical map table consists of 3 single table:
    • TBICMAPTABLES holds the link to the original mapfile.
    • TBICMAPTABLECOLUMS holds the headlines of the mapfile
    • TBICMAPDATA holds the detailed information of the mapfile itself.
