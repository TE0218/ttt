BIC Repository - Tutorial

Instruction

Illustration

  1. Import a project for repository export. 

Mapping Designer - Mapping Import

  1. Select and import the project file from class material.

  2. Choose the proposed value for the project name.

Mapping Designer - Import Project

  1. Select all at the left side and confirm with OK..

Mapping Designer - Mapping Import

  1. Create a repository connection.

Mapping Designer - Repository Connection

  1. Enter the repository server name. Your trainer will tell you the hostname to connect (hostname of the trainers BIS system).

The trainer has to start his BIS system before you will be able to connect to his repository.

Mapping Designer - Repository Server

  1. Login with your login information and confirm the success message.

Logical System: 000

User: superadmin

Password: seeburger

Mapping Designer - Repository Log-on

  1. Select the project and select 'Repository' - 'Export to repository' in the menu you enter with right-mouse-click.

Mapping Designer - Repository Export

  1. Select  name of your repository location and enter a comment.

Mapping Designer - Repository Export

  1. Select the root of the repository and choose 'New folder'.

Mapping Designer - Repository Export

  1. Enter the name of the new folder.

The folder must be unique. So please add the number of your academy user name as extension.

Mapping Designer - Repository Export

  1. Select this new folder and confirm your choice here and in the next mask with OK.

Mapping Designer - Repository Export

  1. Confirm that all objects will exported to the repository.

Mapping Designer - Repository Export

  1. The exported project is now red colored and all mapping get an initial version number 'V0000'.

Mapping Designer - Repository Export

  1. Check-out this mapping right clicking on the mapping.

Mapping Designer - Repository Check-out

  1. Select no message for this check out.

  2. Confirm the Check out success message.

Mapping Designer - Repository Check-out

  1. Insert a traceln-command in this mapping and save your change.

  1. Check-In and activate this mapping..

Mapping Designer - Repository Check-in and activate

  1. Insert a comment and confirm the ckeck-in process..

Mapping Designer - Repository Check-in and activate

  1. Enter a comment for the activation and confirm.

Mapping Designer - Repository Check-in and activate

  1. Repeat the check-out steps 16-18.

  2. Choose 'Import from repository' in the repository menu.

Mapping Designer - Repository Import

  1. Select the name of the repository location.

Mapping Designer - Repository Import

  1. Select the version 0000 from the repository tree.

  2. Confirm your choice.

Mapping Designer - Repository Import

  1. Skip the import of the messages.

Mapping Designer - Repository Import

  1. Overwrite the existing mapping.

Mapping Designer - Repository Import

  1. Confirm the finished import process.
