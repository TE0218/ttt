Tree View

  • Located in the Configuration tab under Converter settings.

  • Shows all mappings and messages in a tree view (comparable to the Mapping Designer).

  • Shows details of all available versions.

  • Delete / purge of mappings / projects

  • Unlock mappings

For more details please refer to Tree View

The Repository Administration GUI's Tree View (shown in the figure below) presents a unified view of all user objects within the BIS Repository:

  • projects
  • messages
  • message versions
  • mappings
  • mapping versions
  • mapping runtime
  • global procedures
  • global procedure versions
  • global procedure runtime

If you click on an object in the tree, the corresponding information will be displayed in the Details panel (right) under several tabs.

Manipulations of an object can be performed from its context menu (available via a right-click) or from the tool bar.

Note: If an operation on an object with children is performed, it also applies to all children. (E.g. deletion of a message forces deletion of all related versions).

The following tables show the type of information provided in the tabs on the right. The available tab types depend on the selected object type.

Format Folder

Usually a set of folders is created automatically when defining a project, one per available EDI format. The folder names correspond to the format names, e.g. ANSI X12, EDIFACT, INHOUSE, etc.
