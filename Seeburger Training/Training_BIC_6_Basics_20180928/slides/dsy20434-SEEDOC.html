The table shows a small subset of all possible umlauts. The Umlaut conversion is performed based on the entries in the resource pool file Charsetb.dta (Umlaut File). For an umlaut expansion the entry behind the comma, for an umlaut ignoring its first character is used. 

If a user specifies his own file for the umlaut conversion via openConvertUmlautFile it overlays Charsetb.dta completely (content is not merged) and the entries are interpreted as in Charsetb.dta.
