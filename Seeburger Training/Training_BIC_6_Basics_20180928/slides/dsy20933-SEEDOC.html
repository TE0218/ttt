BIC and Java

  • BIC is based on a Java Runtime

  • While compiling a mapping / procedure inside the BIC Mapping Designer, there are two steps being executed:

    1. The pre-compiler checks the syntax of the mapping. If no errors are found, a transformation from the SEEBURGER BIC command language into a native Java program takes place (Result: *.java file)

    2. Afterwards the *.java file is compiled by the Java compiler (Result: *.class file).

  • Thus every BIC mapping itself is already platform-independent

  • The functional range of the BIC MD programming language can optionally extended through the integration of native Java code
