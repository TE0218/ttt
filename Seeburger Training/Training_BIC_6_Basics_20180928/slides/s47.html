Overview - 1

There is a large command set which can be divided into different categories:

  • Map commands

    • Used to transfer data from source to destination file

  • Variable declaration

    • Used to create different types of variables to store values during runtime

  • String commands

    • Used to handle string values (cut, extract, search, determine length ...)

  • Numeric commands

    • Used to format numeric values or do calculations

  • Date and Time commands

    • Used to edit and format date and time values
