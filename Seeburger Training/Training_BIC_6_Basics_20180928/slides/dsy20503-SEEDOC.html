EDIFACT BIC Commands

  • The UNT segment contains the number of segments in the message. The UNZ segment contains the number of messages in an interchange file.

  • BIC offers the functionality to automatically count and set the values when creating an EDIFACT message. The easiest way is to use the field properties (tab Codes & counter)

    • Segment counter in UNT.0074

      • Counted record: ALL / Reset records: UNH

    • Message counter in UNZ.0036 (see example)

      • Counted record: UNH /Reset records: UNB

For more details please refer to

There is also a possibility to reset the counters via mapping code:

  • UNT.0074: ResetCounter(UNH,ALL)
  • UNZ.0036: ResetCounter(UNB,UNH)
