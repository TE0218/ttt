Variables

  • There are different types of variables:

Suffix

Type

Example

$

Alphanumeric variable

text$

%

Numeric variable

number%

?

Boolean variable

flag?

~

Big decimal variable

number~
  • Before using a variable you must declare it by defining its type and required lifespan

Declaration

Lifespan

Example

global

Duration of the complete source file / conversion

global text$;

dim

Duration of a source message

dim number%;

local

Duration of the record program

local flag?;
