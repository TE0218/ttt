Map commands

The basic function of a mapping is to move data from a source to a destination. This is done by a couple of commands which can be divided into direct and indirect commands as shown below:

Direct commands:

  • copy: default command to transfer a source value to the destination message (existing values will not be overwritten)

  • existCopy: checks if a source value exists before the value is copied to the destination field

  • overwrite: command which overwrites existing fields

  • existOverwrite: checks if a source value exists before the destination value is overwritten


Indirect commands (allow the usage of a address path built during runtime):

  • setField (address, value): writes a given value to the provided field address

  • getSourceField (address): returns the value of the provided field address
