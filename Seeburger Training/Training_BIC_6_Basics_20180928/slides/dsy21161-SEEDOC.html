Mapfile/MapDB - Tutorial 2

Instruction

Illustration

  1. Open the channels for the mapfile..

 
// Open Mapfile Channels
openMapFile("EAN","INTERNAL_ARTICLE_NUMBER_TO_EAN", 
            1, 2,";","unknown");
openMapFile("description","INTERNAL_ARTICLE_NUMBER_TO_EAN", 
            1, 3,";","unknown");

  1. Define variables, set fix value '123546'.

  2. Insert the mapfile command.

  3. Insert traceln-commands for the output variables.
 
local internal_article_number$;
local EAN$;
local description$;
internal_article_number$ = "123546";

// Use Mapfiles
copy mapfile("EAN",internal_article_number$)   to EAN$;
copy mapfile("description",internal_article_number$)  to description$;

traceln("EAN         : " & EAN$);
traceln("Description : " & description$);
  1. Import the mapfile using the import function of BIC Mapping Designer.

mapfile = classmaterial
separator = ;
character set =
overwrite = true
headline = false

For versions older 6.3.5 Q4 you have to use the batch script \bin\run.mapfileimport... for the import.
